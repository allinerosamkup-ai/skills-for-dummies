# 🤖 SKILL FOR DUMMIES (SURGE CORE v3.0 - Mastery Upgrade)

## 🎯 Propósito
Auto-evolução de elite: Gestão de patrimônio sistêmico, memória técnica de tarefas efêmeras e garantia de integridade via validação de impacto.

---

## 🛠️ Tool Integration
Esta skill possui autoridade para:
1. `list_directory`: Auditoria de skills e projetos.
2. `write_file`: Materializar skills CORE, registrar Snippets e atualizar o System Map.
3. `read_file`: Analisar dependências e validar logs de erro.
4. `run_shell_command`: Executar testes de regressão após Patches.

---

## 🧠 Memória e Integridade (V.I.P.S)
1. **[V]alidação Ativa**: Todo Patch em skill existente exige um teste de funcionamento.
2. **[I]mpacto (System Map)**: Mapear quais projetos dependem de quais skills.
3. **[P]atrimônio vs. Efêmero**: CORE vira Skill; Tarefas viram Snippets.
4. **[S]nippets (Knowledge Base)**: Lógicas úteis são guardadas como código reutilizável, sem poluir o diretório com pastas.

---

## 🧪 Critério de Sucesso
- Skills CORE puras e funcionais.
- `snippets.md` atualizado com resoluções de tarefas efêmeras.
- `system-map.json` refletindo a arquitetura real do workspace.
