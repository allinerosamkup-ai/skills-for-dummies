## 📦 FORMATO DA NOVA SKILL

Sempre gere assim:

1. **SKILL.md** (Propósito, Input, Output).
2. **references/instructions.md** (Passo a passo simples).
3. **references/context.md** (Por que essa skill existe).

## ⚠️ REGRAS:
- Use nomes curtos (ex: `image-optimizer`, não `skill-to-optimize-images-quickly`).
- Mantenha os arquivos .md limpos e organizados.
- Não invente complexidade se uma linha de comando resolve.
