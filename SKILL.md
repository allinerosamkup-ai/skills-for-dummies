# 🤖 SKILL FOR DUMMIES (SURGE CORE v2.0)

## 🎯 Propósito
Auto-evolução simplificada: detectar lacunas no sistema e materializar novas skills funcionais sem "juridiquês" ou complexidade desnecessária.

---

## 🛠️ Tool Integration
Esta skill possui autoridade para:
1. `list_directory`: Consultar `skills/` para evitar redundância.
2. `write_file`: Criar pastas e arquivos `.md` da nova skill.
3. `read_file`: Diagnosticar falhas de capacidade em logs ou conversas.

---

## 🧠 Modo de Operação (D.U.M.M.Y)
1. **D**etectar: O problema é falta de instrução ou erro de código?
2. **U**nificar: Juntar contexto e comportamento esperado.
3. **M**aterializar: Escrever os arquivos da nova skill no disco.
4. **M**odelar: Ajustar o padrão para ser simples e reutilizável.
5. **Y**es! (Aprovar): Validar se a skill resolve o problema original.

---

## 🧪 Critério de Sucesso
- Nova skill materializada em `skills/[nome-da-skill]/`.
- Instruções diretas e funcionais.
- Zero conflito com skills existentes.
